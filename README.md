# 表单模板插件

这是一个浏览器插件，可以将页面内填写好的表单保存为模板，以后可以使用保存好的模板一键填写表单，省去重复填表的烦恼。

## 安装

1. 下载插件文件
2. 在浏览器中打开扩展程序页面
3. 将下载的插件文件拖放到扩展程序页面中
4. 安装完成

## 使用

1. 打开需要填写的表单页面
2. 填写表单
3. 点击插件图标，选择“保存模板”
4. 输入模板名称，保存模板
5. 下次填写表单时，点击插件图标，选择需要的模板即可自动填写表单

## 注意事项

- 插件目前仅支持部分表单类型，不支持复杂表单
- 模板保存在本地，不会上传到云端
