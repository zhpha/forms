# Form Filler - 表单模板插件

一个简单高效的浏览器插件，旨在通过将填写的表单保存为模板，实现一键自动填充，极大减少重复录入的工作量。

## 🚀 主要功能

- **保存模板**：自动识别页面内已填写的表单内容并保存为自定义名称的模板。
- **一键填充**：在相同的表单页面，只需点击一下即可根据保存的模板自动填入所有字段。
- **本地存储**：所有模板数据均保存在浏览器本地，保护隐私，不上传云端，确保安全。
- **多页面支持**：支持大部分常见的 HTML 表单类型。

## 📦 安装方法

您可以按照以下步骤手动安装本插件：

1. **下载源码**：克隆本仓库或下载 ZIP 包并解压。
   ```bash
   git clone https://github.com/zhpha/forms.git
   ```
2. **打开扩展程序页面**：在浏览器地址栏输入 `chrome://extensions/`（Chrome）或 `edge://extensions/`（Edge）。
3. **开启开发者模式**：打开页面右上角的“开发者模式”开关。
4. **加载插件**：点击“加载解压的扩展程序”，选择本项目所在的文件夹。
5. **完成安装**：在扩展列表中看到“表单模板插件”图标即表示成功。

## 🛠️ 使用说明

1. **录入数据**：在表单页面填写好内容。
2. **保存模板**：点击插件图标，选择“保存模板”并输入名称。
3. **自动填充**：再次访问时，点击插件图标并选择对应模板即可。

## ⚠️ 注意事项

- 插件目前支持常见的 `input`, `select`, `textarea` 等表单元素。
- 模板数据保存在本地浏览器缓存中，清除浏览器数据可能会导致模板丢失。
